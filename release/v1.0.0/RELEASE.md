# DOR101 v1.0.0 — Windows Release

**Release date:** June 7, 2026  
**Platform:** Windows 10/11 (64-bit)

## Downloads

| File | Description | Size |
|------|-------------|------|
| `DOR101 Setup 1.0.0.exe` | Full installer with Start Menu and Desktop shortcuts | ~174 MB |
| `DOR101-Portable-1.0.0.exe` | Portable app — no installation required | ~174 MB |

## System Requirements

- Windows 10 or Windows 11 (64-bit)
- 4 GB RAM minimum
- 500 MB free disk space
- Internet connection (for live news, transit, and market data)

## Installation (Setup)

1. Download `DOR101 Setup 1.0.0.exe`
2. Double-click to launch the installer
3. Choose an installation directory (or accept the default)
4. Click **Install**
5. Launch **DOR101** from the Desktop shortcut or Start Menu

## Portable Usage

1. Download `DOR101-Portable-1.0.0.exe`
2. Place the file anywhere (USB drive, Desktop, etc.)
3. Double-click to run — no admin rights or installation needed

## Notes

- Windows SmartScreen may show an "Unknown publisher" warning because the app is not code-signed. Click **More info → Run anyway** to proceed.
- No account, API key, or database setup is required for the desktop app.
- All preferences (language, theme, font size) are stored locally on your device.
